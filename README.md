# TECH-SY 🚀

A modern digital solutions landing page built using **HTML and CSS**. This project showcases different tech services with a clean design and responsive layout.

---

## 📌 About the Project

TECH-SY is a simple and modern website designed to present digital services like web development, cybersecurity, AI integration, and cloud solutions. It focuses on clean UI, structured sections, and a smooth user experience.

---

## ✨ Features

* Clean and modern UI
* Fully responsive design
* Well-structured sections
* Service showcase cards
* Simple navigation (Home, About, Contact, Account)
* Search bar UI

---

## 🛠️ Tech Stack

* HTML
* CSS

---

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/CodeWithVishal-18/TECH-SY.git
```

2. Open the project folder

3. Run the project

* Open `index.html` in your browser
